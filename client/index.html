Realtime Data

With VerisDB Analyst, you can search the dataset in realtime because the application is isomorphic. If you have a rapidly changing dataset, you can easily view changes in realtime.

Comprehensive Charting

The charts included with VerisDB Analyst are comprehensive and provide a broad view of any VERIS dataset in realtime. The official dataset is included by default.

Advanced Search Capability

VerisDB Analyst supports direct MongoDB search queries and so data can be searched and filtered with tailored searches.
